# Sketch on Thoughts – Architecture

## Run in the browser

1. **Install dependencies** (first time only):
   ```bash
   npm install
   ```

2. **Start the dev server**:
   ```bash
   npm run dev
   ```

3. **Open in browser**  
   In the terminal you’ll see something like:
   ```text
   Local:   http://localhost:5173/
   ```
   Open that URL in your browser (Chrome, Edge, Firefox, etc.).

The page will reload when you edit the code. To stop the server, press `Ctrl+C` in the terminal.

---

**Build for production:** `npm run build`  
**Preview production build:** `npm run preview`
